This is a front-end project that replicates the visual design and layout of the Amazon homepage. It was created as a practice project to apply and strengthen my skills in modern HTML5 and CSS3.

## ‚ú® Key Features

This clone includes the following sections and features:

- **Navigation Bar:** A faithful recreation of the multi-level Amazon navigation bar, including the search box, account links, and cart.
- **Hero Section:** The main "hero" banner component.
- **Product Grid:** A responsive grid layout displaying product cards, similar to Amazon's "Today's Deals" or "Recommended" sections.
- **Footer:** The complex, multi-column footer with navigation links.
